# Notas de Atletas

## Descrição
Sistema para gerenciar e acompanhar notas de desempenho de atletas.

## Funcionalidades
- Registrar atletas
- Adicionar notas de desempenho
- Visualizar histórico de notas
- Gerar relatórios

## Instalação
```bash
git clone <seu-repositorio>
cd notas-atletas
npm install
```

## Uso
```bash
npm start
```
